Windows is an operating system first released by Microsoft in 1985. Since then, it has gone through 10 major updates to increase usability and add new features. Some of these updates, namely Windows 98 and Windows Vista, did not do well when released to the community. Windows 95, Windows XP, and Windows 7 were all strong updates for the operating system that helped it to secure its spot in the market today.
